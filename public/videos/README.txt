This directory is for video files used in the application.

For the hero background, you should add a video file named "food-background.mp4" here.
This video should be a high-quality, looping video of food preparation or food truck ambiance.

Recommended specifications:
- Resolution: 1920x1080 or higher
- Duration: 10-30 seconds (will be looped)
- Format: MP4
- Codec: H.264
- File size: Keep under 5MB if possible for faster loading

You can find suitable stock videos on sites like:
- Pexels (https://www.pexels.com/videos/)
- Pixabay (https://pixabay.com/videos/)
- Unsplash (https://unsplash.com/videos)
